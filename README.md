🏛️ La Lucentina | E-commerce Success Case
Estado: 🟢 En Producción | Cliente: Comercio Local (Sector Alimentación/Artesanía)

Este no es un proyecto de aprendizaje más; es una solución real desplegada para La Lucentina, con el objetivo de digitalizar su catálogo y optimizar la captación de clientes locales.

<img src="/Portada.png" alt="Portada de la web"></img>

🎯 El Reto de Negocio
El cliente necesitaba una presencia digital profesional que permitiera a los usuarios explorar su catálogo de productos de forma ágil, segura y con un diseño que reflejara la identidad de la tienda. El objetivo principal: rendimiento extremo y seguridad de datos.

🛠️ Stack Tecnológico y Decisiones Técnicas
Para garantizar la escalabilidad y la mantenibilidad, elegí un stack moderno y tipado:

Frontend: React + TypeScript. El uso de TS fue innegociable para asegurar la integridad de los datos en el carrito de compras y la gestión de productos.

Estilos: Tailwind CSS. Implementación de un diseño mobile-first con tiempos de carga mínimos.

Backend & DB: Supabase (PostgreSQL). Elegido por su capacidad de respuesta en tiempo real y su robusto sistema de autenticación.

🚀 Puntos Clave de Ingeniería (Lo que me diferencia)
1. Seguridad Avanzada con RLS
Implementé Row Level Security (RLS) en PostgreSQL a nivel de base de datos. Esto garantiza que, aunque el frontend sea accesible, nadie puede modificar precios o stock sin las políticas de acceso correctas, moviendo la lógica de seguridad al motor de la base de datos.

2. Rendimiento & SEO
Imágenes optimizadas para carga progresiva.
Puntuación de +90 en WCAG en accesibilidad y mejores prácticas.
Arquitectura de componentes reutilizables para facilitar futuras expansiones del catálogo.
